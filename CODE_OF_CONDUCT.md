# PyLadies Shizuoka Code of Conduct

版：1.0  
制定日：2026-03-22  
最終更新日：2026-03-22  

---

本コミュニティは、すべての参加者が安心して参加できる環境を大切にします。

## 行動指針

- 相手を尊重する
- 多様性を受け入れる
- 威圧的・攻撃的な言動をしない
- 初心者の「わからない」を大切にする

## 禁止事項

- ハラスメント行為
- 差別的な発言
- 他者を不快にさせる行為

## 対応

問題が発生した場合、運営が適切に対応します。
